# sharemeshi
Sharemeshi is a food sharing platform for home made food lovers. Visit our [official website](https://sharemeshi.com/) to learn more!

#### This app is bootstraped by AngularJS.

### Technology stack
`AngularJS + Node.js + MongoDB `
